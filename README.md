# Instructions

## Installation
Download `Graph.jar` or clone this repository

## Execution
 - `jar -xf Graph.jar`
 - `java -jar Graph.jar`

## Execution of the implementation
 - `jar xf graphImpl.jar` -> Extract files 
 - `java -jar graphImpl.har` -> Execute the implementation
